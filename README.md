# Litorale

Personal playground for [Lit library](https://lit.dev/). 

## Package manager

Used pnpm for the first time

## Bundler

The entire project was bootstraped using `vite create` with template `lit-ts`

## Lint

```bash
npm run lint:fix 
```

## Format

```bash
npm run format:fix
```

## Test

```bash
npm run test
```

watch

```bash
npm run test:watch
```
